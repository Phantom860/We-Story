# 友谊记录 - Friendship Memory

一个温馨的友谊记录应用，帮助你记录与朋友之间的美好时光、重要纪念日、心愿清单和珍贵照片。

## ✨ 功能特性

### 📔 日记记录
- 记录每一天与朋友相处的点滴
- 支持心情标记（开心、难过、平静、兴奋）
- 标签分类管理
- 标记特别记忆

### 🎂 纪念日提醒
- 添加重要的友谊纪念日
- 自动计算距离纪念日的天数
- 支持年度重复提醒
- 纪念日倒计时展示

### 💝 心愿清单
- 创建共同的心愿目标
- 优先级排序（高、中、低）
- 分类管理（浪漫、旅行、美食、其他）
- 状态跟踪（待实现、进行中、已完成）

### ✅ 打卡清单
- 记录待完成事项
- 完成状态切换
- 进度统计展示
- 负责人分配

### 🖼️ 相册功能
- 上传和管理照片
- 照片搜索（按标题、描述、标签）
- 照片详情展示
- 地点和日期标记

## 🛠️ 技术栈

- **框架**: React 19 + TypeScript
- **构建工具**: Vite 6
- **样式**: Tailwind CSS 3 + Radix UI
- **图标**: Lucide React
- **路由**: React Router
- **数据存储**: LocalStorage（本地持久化）

## 🚀 使用说明

### 日记记录
1. 点击「写日记」按钮打开日记编辑表单
2. 填写标题、内容，选择心情和标签
3. 勾选「特别记忆」标记重要时刻
4. 点击「保存日记」完成记录

### 纪念日提醒
1. 点击「添加纪念日」按钮
2. 输入纪念日名称和日期
3. 选择是否每年重复提醒
4. 系统自动计算倒计时天数

### 心愿清单
1. 点击「新增心愿」按钮
2. 填写心愿标题和描述
3. 设置优先级和分类
4. 通过「进行中」「已完成」按钮更新状态

### 打卡清单
1. 点击「新增事项」按钮
2. 填写事项标题和描述
3. 设置截止日期和负责人
4. 点击复选框标记完成状态

### 相册管理
1. 点击「上传照片」按钮
2. 选择本地图片文件（或使用默认图片）
3. 填写照片标题、描述、地点和标签
4. 点击「上传照片」完成上传

## 🔒 数据存储

应用使用浏览器的 LocalStorage 进行数据持久化，所有数据存储在本地，不会上传到服务器。

数据存储键名：
- `__global_friendship_diaries` - 日记数据
- `__global_friendship_anniversaries` - 纪念日数据
- `__global_friendship_wishes` - 心愿数据
- `__global_friendship_checklists` - 打卡清单数据
- `__global_friendship_photos` - 照片数据
